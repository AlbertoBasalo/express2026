---
name : init-express
description : Initialize a new Express.js project with TypeScript, Biome, Vitest, and Playwright.
---

Run the following commands to set up a new Express.js project with TypeScript, Biome, Vitest, and Playwright:
```bash
git init
npm init
npm i -D @types/node
npm i -D typescript tsx
npx tsc --init
npm i express@latest
npm i -D @types/express@latest
npm i -D @biomejs/biome
npm i -D vitest
npm init playwright@latest
npx playwright install
```
